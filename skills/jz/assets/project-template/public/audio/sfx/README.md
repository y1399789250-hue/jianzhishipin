Save short sound effects here.

Recommended categories:

- `paper-tap.wav` for source cards or paper labels sticking to the scene.
- `paper-pop.wav` for light object pop-ins.
- `soft-land.wav` for characters or objects settling on the ground.
- `page-swipe.wav` for scene transitions.
- `small-tick.wav` for small emphasis beats.

Use `scene.sfxEvents` in `src/script.json` for precise timing.
