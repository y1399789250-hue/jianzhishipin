# 验收清单

## 文案和镜头

- [ ] 每个镜头只讲一个叙事点。
- [ ] 每个镜头都有明确主角。
- [ ] 镜头之间有推进关系，不是同一画面的重复变化。
- [ ] `src/script.json` 中 `durationSec` 与旁白长度基本一致。

## 素材

- [ ] 背景底板没有主要人物。
- [ ] 每个可动人物都是独立 PNG。
- [ ] 透明 PNG 没有明显绿边、白边断裂或残留背景。
- [ ] 人物头、手、脚完整，没有被裁掉。
- [ ] 朝向符合镜头关系。
- [ ] 素材路径全部相对 `public/`。

## 静态排版

- [ ] 主角最大且第一眼能被看见。
- [ ] 配角没有挡住主角脸、手、关键道具。
- [ ] 后排人物不会抢戏。
- [ ] 前景遮挡能制造纵深，不遮挡重点。
- [ ] 人物脚底落点合理。
- [ ] 字幕区没有和人物脚部、道具冲突。

## 动画

- [ ] 主角、配角、后排人物错峰出现。
- [ ] 背景只有轻微推镜或漂移。
- [ ] 主角运动幅度最大，后排运动幅度最小。
- [ ] 入场后仍有很轻的漂浮，不是完全静止。
- [ ] 没有人物突然闪现或同时塞满画面。

## 音频

- [ ] 旁白清晰，无明显底噪和音乐泄漏。
- [ ] 背景音乐低于人声。
- [ ] 主角入场音效比配角更有力量。
- [ ] 音效和入场帧对齐。
- [ ] 字幕和旁白句子对应。

## 技术验收

- [ ] `npm run validate` 通过。
- [ ] `npm run validate:strict` 通过。
- [ ] `npm run probe` 能看到正确分辨率、时长和音轨。
- [ ] `npm run frame:check` 抽出的检查帧无明显问题。
- [ ] 最终 MP4 能正常播放。
